API 概述

各种应用整合 baigo SSO 都是通过 API 接口实现的，您可以在各类应用程序中使用该接口，通过发起 HTTP 请求方式调用 baigo SSO 服务，返回 JSON 数据。

使用 API 接口，您需先在 baigo SSO 创建应用，创建成功后会给出 APP ID 和 APP KEY。详情查看 查看应用。

应用的验证

baigo SSO 的所有 API 接口均需要验证应用以及验证应用的权限。详情请参照具体接口说明。

返回结果

baigo SSO 所有 API 接口均返回加密字符串，真正内容需要调用密文接口进行解密。

返回结果
名称 类型 具体描述
str_alert string 结果代码，详情请看 结果代码。
code string 加密字符串，需要解密。详情请看 密文接口。
key string 解密码，配合加密字符串使用，用于解码。详情请看 密文接口。
